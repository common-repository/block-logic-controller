Add block criteria or conditions to show or hide or even replace them.

== Description ==
This plugin shows and hides block based on several criteria or conditions set to it. If criteria is matched, the block will be shown or hidden based on selection action.

<h3>Key Features & Options</h3>

Logic actions:
<ul>
    <li>Show block.</li>
    <li>Hide block.</li>
    <li>Replace block.</li>
</ul>

Logic criteria:
<ul>
    <li>User logged in/out.</li>
    <li>Using mobile device.</li>
    <li>User roles.</li>
    <li>User metakeys.</li>
</ul>
